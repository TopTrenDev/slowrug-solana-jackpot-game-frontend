Design system for SlowRUG - Solana jackpot game

- Theme: Multi-neon cyberpunk combining 4 palettes (Midnight Gold, Blood & Chrome, Toxic Terminal, Deep Ocean)
- Background: Deep navy (225 45% 7%) — from Midnight Gold
- Primary: Acid yellow-green (72 100% 50%) — from Toxic Terminal, used for CTAs, wins, active states
- Secondary: Electric cyan (187 100% 50%) — from Deep Ocean, used for secondary highlights
- Accent: Warm gold (43 78% 46%) — from Midnight Gold, used for premium/earnings elements
- Destructive: Hot red (0 100% 59%) — from Blood & Chrome, used for loss/danger states
- Chrome: Silver (220 10% 75%) — from Blood & Chrome, used for subtle UI text
- Fonts: Bebas Neue (headings/display), Space Grotesk (body), IBM Plex Mono (data)
- Buttons: Sharp/angular with clip-path cuts, no rounded-2xl, uppercase tracking
- Pages: Home, Tower, Infinite Rug, Operators Hub
- Layout: Collapsible sidebar + particle background + live chat overlay
- Effects: Neon glows (green/cyan/gold/red), scanlines, gradient borders, particle field canvas
- Chat: Live player chat (floating bottom-right bubble)
- Images: AI-generated cyberpunk artwork for hero + game banners
- No real blockchain - UI only with mock data
- Use semantic tokens (text-primary, bg-card, etc.) not raw hsl in components
